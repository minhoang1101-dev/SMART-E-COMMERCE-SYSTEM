# Smart Ecommerce System Web App

This adds a browser-based frontend and a small Node.js backend to the existing C++ text-file data project.

## Requirements

- Node.js 18 or newer

## Run

```powershell
npm start
```

Open:

```text
http://localhost:3000
```

## Demo Logins

- Admin: `admin` / `admin123`
- Customer: `minhoang` / `123456`

## What Is Included

- Product browsing and search
- Sign in, sign up, and guest access
- Admin product creation and deletion
- Product view tracking
- Local cart in the browser
- Checkout and order history
- Simple recommendations based on product views and purchases

## Project Structure

```text
backend/server.js     Node.js API and static file server
frontend/index.html   Web page
frontend/styles.css   Page styling
frontend/app.js       Browser behavior
data/*.txt            Existing text-file storage
```
